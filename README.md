# Global Checkout System

This project demonstrates the Factory Method pattern in a Node.js checkout system. You can test credit card, PayPal, and Bitcoin payment flows using a simple command-line menu.

## How to Run

1. **Install Node.js** if you don't have it yet:  
   [Download Node.js](https://nodejs.org/)

2. **Clone or download this repository** to your machine.

3. **Open a terminal in the project folder and run:**  
   ```
   npm install
   ```

4. **Start the checkout system:**  
   ```
   npm run checkout
   ```
   or
   ```
   node main.js
   ```

5. **Follow the interactive menu in your terminal:**  
   - Type `1` for Credit Card
   - Type `2` for PayPal
   - Type `3` for Bitcoin
   - Type `4` to test all payment methods
   - Type `0` to exit
